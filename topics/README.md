# Mathematics Münster — Topic Group Deep Dives

Detailed companion files to [`../EXPLORATION.md`](../EXPLORATION.md). Each file
covers one of the ten Mathematics Münster **topic groups**: the topic's objective,
then its core mathematical concepts explained at **three levels**:

- **Level 0** — high school student
- **Level 1** — undergraduate (bachelor) mathematician
- **Level 2** — master's-level mathematician

Source: [MM Research Programme](https://www.uni-muenster.de/MathematicsMuenster/research/programme/index.shtml)

## Notes on formatting

- Mathematics is written in GitHub/LaTeX style (`$...$` inline, `$$...$$` display).
  GitHub renders this natively; in other viewers use a Markdown+KaTeX/MathJax
  previewer.
- Images are hot-linked from the Mathematics Münster website (© MM/vl) and require
  an internet connection to display.
- Web links point to original publications (DOI/arXiv) and to encyclopaedic
  references for each concept.
